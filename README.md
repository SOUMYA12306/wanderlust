# 🌍 Wanderlust — Travel & Property Listing Platform

A full-stack Airbnb-style web application where users can explore, list, and book travel properties worldwide.

## 🚀 Live Demo
> Deployed on AWS

## ✨ Features
- 🔐 Secure user authentication with JWT
- 🏠 Dynamic property listing & booking
- 🖼️ Property image upload via Cloudinary
- 📱 Fully responsive design with Tailwind CSS
- ⚡ Optimised media handling — 35% faster page load
- ☁️ Deployed on AWS for scalability

## 🛠️ Tech Stack
| Frontend | Backend | Database | Cloud |
|----------|---------|----------|-------|
| React.js | Node.js | MongoDB  | AWS   |
| Tailwind CSS | Express.js | | Cloudinary |
| Axios | JWT | | |

## ⚙️ How to Run Locally
```bash
git clone https://github.com/SOUMYA12306/wanderlust
cd wanderlust
npm install
npm start
```
